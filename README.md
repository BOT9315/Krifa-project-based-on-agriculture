## 🌟 Why Krifa?

Krifa is a cutting-edge SaaS platform designed specifically for modern farmers and agricultural professionals. Our AI-driven solutions help you:

- **Optimize Crop Yields**: Get personalized recommendations based on soil health, weather patterns, and historical data
- **Streamline Operations**: Manage your entire farm from planting to harvest with intuitive dashboards
- **Maximize Profits**: Access real-time market prices and connect directly with buyers
- **Stay Ahead**: Leverage IoT integration and predictive analytics for proactive decision-making

Whether you're managing a small family farm or overseeing large-scale agricultural operations, Krifa provides the tools you need to thrive in today's competitive farming landscape.

## 🚀 Quick Start

### Prerequisites
- Node.js 18+ or Bun 1.0+
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/nayiswftw/Krifa.git
   cd krifa
   ```

2. **Install dependencies**
   ```bash
   # Using Bun (recommended)
   bun install

   # Or using npm
   npm install
   ```

3. **Start the development servers**
   ```bash
   # Start both frontend and backend
   bun run dev

   # Or start individually
   bun run dev:client    # Frontend on port 8080
   bun run auth-server   # Backend on port 3000
   ```

4. **Open your browser**
   Navigate to `http://localhost:8080` to access the Krifa dashboard.

### Build for Production

```bash
# Create optimized production build
bun run build

# Preview production build locally
bun run preview
```
